
# <p align="center">Nodecomm</p><p align="center">

## Description

An e-commerce solution using TypeScript and Nest.js.
Main Stack: TypeScript, Nest.js, PostgreSQL, GraphQL, OAuth, Passport.js, Jest

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
